# Style Guards

## Bom tom

- claro
- humano
- especifico
- confiante sem arrogancia
- profissional sem corporativismo vazio

## Hook forte

Prefira abrir com:
- contraste
- aprendizado inesperado
- numero concreto
- tensao real
- frase opinativa defensavel

## Evite

- "Passando aqui para compartilhar"
- "Tive o prazer de"
- "Mais um projeto incrivel"
- frases genéricas de superacao
- adjetivos vazios como inovador, disruptivo, transformador sem prova

## CTA

Use CTA apenas se ajudar:
- conversa
- demonstracao
- candidatura
- feedback
- compartilhamento

## Hashtags

Use zero a tres.
So mantenha se ajudarem distribuicao ou contexto.

## Regra final

Se o texto puder servir para qualquer pessoa, ele ainda esta generico demais.
